# Veldra Textiles — Technical Luxury Website

## 🌟 Overview

Veldra Textiles is a premium luxury textile brand featuring an elegant, modern website showcasing technical innovations in fabric design and sustainable manufacturing.

### Key Features

✨ **Premium Design**
- Sleek dark mode with gold accents
- Custom animated cursor
- Smooth scroll reveal animations
- Responsive grid layouts

🎨 **Sections**
- **Hero**: Captivating introduction with call-to-action
- **Studio**: About Veldra's founding and mission
- **Collection**: 5 signature fabric collections with detailed specs
- **Technology**: 5 core technologies powering the fabrics
- **Stats**: Key metrics and trusted partners
- **Process**: 5-phase fabric production timeline
- **Testimonials**: Client reviews and quotes
- **Sustainability**: Environmental commitments and certifications
- **Contact**: Commission inquiry form
- **Footer**: Navigation and legal info

🎯 **Interactive Elements**
- Animated marquee of product names
- Hover effects on collection cards
- Custom cursor with ring animation
- Scroll-triggered reveal animations
- Form submission feedback

## 🛠️ Technologies Used

- **HTML5**: Semantic structure
- **CSS3**: Grid, flexbox, animations, gradients
- **JavaScript**: Vanilla JS for interactivity
- **Google Fonts**: Bebas Neue, Libre Baskerville, Barlow

## 📱 Responsive Design

- Mobile-first approach
- Breakpoint at 900px for tablets/desktop
- Adaptive grid layouts
- Touch-friendly interactive elements

## 💡 Customization

### Change Colors
Edit the CSS variables in the `:root` selector:
```css
:root {
  --gold: #C9A84C;
  --black: #080808;
  /* ... more colors ... */
}
```

### Modify Content
All text content is in the HTML — simply edit the text within tags:
```html
<h1>Your New Title Here</h1>
```

### Update Contact Info
Find the contact section and update addresses, emails, and phone numbers.
